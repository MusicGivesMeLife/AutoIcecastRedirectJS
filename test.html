Android Devices
Later versions of Android can use the following URL: http://ic2.christiannetcast.com/westpca 
Please press "Play" if the player is paused. Stream should start within 20 seconds.

For PC Users
Clicking the link above should ask you to download an m3u file. After you do so, double click the file and the stream should open and begin playing immediately. You don't need to download a new file each time you tune in; simply open up the same file you downloaded the first time.
Note: If the stream opens in Windows Media Player and you don't hear anything, try clicking on the play/pause bar as circled in red here.
Can't Use Windows Media Player?
VideoLAN VLC is probably the best option for users of computers (Windows XP SP2 or later, and many other operating systems) that cannot use Windows Media Player. Download it here:http://www.videolan.org/vlc/.
For MAC Users
The stream should begin automatically when you click the link.
